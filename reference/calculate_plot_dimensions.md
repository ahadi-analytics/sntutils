# Calculate appropriate plot dimensions based on data

Calculate appropriate plot dimensions based on data

## Usage

``` r
calculate_plot_dimensions(plot_data, x_var, y_var = NULL)
```

## Arguments

- plot_data:

  Data frame with the plotted data

- x_var:

  X-axis variable

- y_var:

  Y-axis variable (can be NULL)

## Value

List with width and height values
