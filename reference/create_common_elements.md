# Create common ggplot elements

Create common ggplot elements

## Usage

``` r
create_common_elements(
  fill_var,
  fill_limits,
  use_reprate = TRUE,
  target_language = "en",
  fill_label = NULL
)
```

## Arguments

- fill_var:

  Fill variable name

- fill_limits:

  Limits for the fill scale

- use_reprate:

  Whether to use reporting rate colors

- target_language:

  Target language for translation

- fill_label:

  Label for the fill legend

## Value

List of ggplot elements to apply to plots
